# 广医小助手

旨在方便广医人日常学习交流的微信小程序

---

## 项目介绍：

一个基于 微信小程序 、express (后端) 的工具类小程序。

## 项目实现：

- [x] 成绩查询系统
- [x] 生日榜单系统
- [X] 用户登录系统
- [X] 文章精选系统

## 项目优势：

- 好看的页面和交互效果(大雾
- 获得近期生日的同学名录
- 每日实时更新的精选文章阅读
- 比官方更早的获知考试成绩
- 获知某学科成绩的排名与百分比
